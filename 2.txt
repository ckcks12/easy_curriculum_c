# 2. 전설의 레전드 Hello, World! 해보기

>Hello, World!

이게 왜 전설의 레전드이냐구요? [위키피디아](https://en.wikipedia.org/wiki/%22Hello,_World!%22_program)에 따르면 *The C Programming Language*에서 예시 프로그램으로 *Hello, World!* 라는  문장을 출력시켰다고 해요. 그리고 이 예시 프로그램이 여러가지 언어를 가르칠때 계속 이용되었답니다. 더 좋은 방법이 있을지도 모르지만 저도 그렇게 배웠고 여러분도 앞으로 그렇게 다른 분들을 가르쳤으면 좋겠습니다. 다른 이유는 없어요. 그냥 재밌잖아요!

그럼 우리 이제 C언어로 *Hello, World!* 를 출력해 볼까요?
```C
#include <stdio.h>
int main()
{
    printf("Hello, World!\n");
}
```
이 소스는 가장 간단한 소스에요. 괄호 `{`, `}` 같은 건 무시하고 핵심 문장만 3줄이나 되네요. 
>안돼요! 벌써 포기하고 뒤로가기를 누르지 말아요

차근 차근 하나씩 가르쳐 드릴게요.

### `#include <stdio.h>`
include 는 *포함하다* 라는 뜻이에요. `<stdio.h>` 라는 것을 포함하네요. *stdio.h* 는 *Standard Input and Output* 이라는 뜻이에요. *기본 입출력* 을 위한 필요한 소스들이 들어있어요. 당연히 우리는 그 소스를 알 필요가 **전혀** 없습니다. 이외에 `#` 과 `< >`는 무엇인지는 넘어가도록 해요. *언어니까 하나의 규칙이라고 생각해요.*

### `int main() {}`
여기서 알아야 할 것은 `int` 가 뭔지가 아니라 `main` 이에요. `main` 이 중요한 이유는 **무조건 main이어야 하기 때문** 이에요. C언어는 사람이 만든 언어여서 위에서 아래로, 왼쪽에서 오른쪽으로 읽습니다. 그런데 컴파일러가 번역할 때에는 컴퓨터 언어로 **무조건 첫 번째 줄부터 실행되도록** 번역하지 않아요. `main` 이라는 곳을 찾아서 그 곳부터 실행되도록 한답니다. *이것도 언어니까 하나의 규칙이라고 생각해요.*

### `printf("Hello, World!\n");`
드디어 가장 의미있어 보이는 곳에 도착했어요. 그런데 가만 보니 **print** 가 아니라 **print`f`** 네요. 오타가 아니에요. **f** 는 *format* 의 약자로 *서식(형식)에 맞추어 출력한다* 라는 뜻이에요. 무슨 서식이 있는지 변수를 배울 때 가르쳐 드릴게요. 

직접 코드를 써보셔서 실행 `Ctrl+B` 시켜보세요! 이리 저리 바꿔보며 소스 코드를 다루어 보는 것이 프로그래밍 실력을 키우는데에 **단언컨대 가장 완벽한 방법** 입니다.
